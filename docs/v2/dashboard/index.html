About this dashboard

This is a simple local dashboard that reads docs/v2/PRODUCT-ROADMAP.md and renders an interactive list. It’s intentionally tiny: no build step, no server required.

To update statuses, edit PRODUCT-ROADMAP.md and refresh (or click Reload data).

Tip: if your browser blocks fetch() for file:// URLs, run a quick local server: 
cd projects/apex-v1/docs/v2/dashboard
python3 -m http.server 7331
# then open http://localhost:7331
